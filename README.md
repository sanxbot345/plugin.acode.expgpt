# ExpGPT - Acode AI Workspace Plugin

ExpGPT adalah plugin kecerdasan buatan revolusioner untuk Acode Editor yang mengintegrasikan AI langsung ke dalam struktur ruang kerja (workspace) Anda. 

## Fitur Utama
1. **Floating Action Button**: Tombol melayang responsif dengan animasi halus, mendukung fitur *drag-and-drop* intuitif di perangkat sentuh.
2. **Glassmorphism UI Panel**: Panel obrolan gelap (*Dark Theme*) modern dengan efek blur latar belakang (*backdrop-filter*).
3. **Multi-Endpoint Support**: Terintegrasi dengan OpenAI, Gemini (OpenAI Compatible), OpenRouter, dan Custom Endpoint.
4. **Workspace Automation**: AI mampu membaca, menulis, mengubah nama, mencari, serta menghapus file dan folder langsung pada proyek aktif Anda menggunakan API FileSystem Acode.
5. **Progress Visualizer**: Indikator progres pengerjaan riil berbasis teks balok (`████░░░░ 40%`) saat AI melakukan operasi file massal.

## Cara Penggunaan
- Sentuh dan geser tombol ExpGPT untuk memindahkan lokasinya.
- Ketuk tombol untuk membuka panel kontrol AI.
- Masukkan konfigurasi API Anda pada tab Pengaturan di dalam panel UI sebelum memulai sesi pengerjaan.
